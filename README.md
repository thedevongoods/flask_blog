# Flask Blog

This flask blog will be based on [Corey Schafer's Flask tutorial](https://www.youtube.com/playlist?list=PL-osiE80TeTs4UjLw5MM6OjgkjFeUxCYH).

#Installation
Download or clone this repo.

Install flask
```bash
pip3 install flask
```

#Usage
```bash
python app.py
```

or 

```bash
export FLASK_APP=app.py
flask run
```
